Detailed Description

A class to represent and construct HTML pages. A page may have 0 to n PageViews (also referred to as subviews) which can be visible or hidden. All subviews are sent to the requesting client as HTML but only certain parts are shown. It is possible to show multiple subviews at once. This class might later be moved to -namespace.
